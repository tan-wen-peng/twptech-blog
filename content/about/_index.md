## 👋 我是谁

TWP，干了十来年工业自动化。PLC 起步，后来转机器视觉，现在 Visual Studio + Halcon 一天不打开就手痒。

这博客没什么高大上的理论分析，就是**产线上真实踩过的坑**。有些坑当时debug到半夜，有些坑客户现场等着，背着一包线缆就飞过去了。

工业现场有个特点：**有些问题书上找不到，老师傅也未必遇到过，只有自己磕一次才能记住。**

这里记下来的，就是那些磕过的。

---

## 💼 技术路线

| 阶段 | 方向 | 工具链 |
|------|------|--------|
| 早期 | PLC + 运动控制 | 汇川 / 三菱 / 西门子，雷赛 / 固高 |
| 中期 | 机器视觉入门 | Halcon + 海康 MVS |
| 现在 | 视觉+上位机全栈 | C# / C++，Halcon .NET，Modbus TCP |

**只会这些：**
- ✅ **机器视觉** — Halcon（主力）、海康 VisionMaster、OpenCV（备胎）
- ✅ **上位机** — C# WinForms / WPF
- ✅ **PLC** — 汇川 H5U/AM600、三菱 FX5U、西门子 S7-1200
- ✅ **工业通信** — Modbus TCP / RTU、EtherCAT、Profinet
- ✅ **打光设计** — 环形光/背光/同轴光/条形光选型布阵
- ✅ **机械设计** — SolidWorks 出图为生，Creo/UG 勉强能看

**绝对不碰：**
- ❌ 前端框架（React / Vue / Node.js — 不熟也不会去学）
- ❌ AI 大模型炼丹
- ❌ Linux 运维 / DevOps
- ❌ 游戏开发

> **主战场：Windows。Linux 用来跑 Docker 装博客都嫌麻烦。**
